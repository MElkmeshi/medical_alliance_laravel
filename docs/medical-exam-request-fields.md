# Medical Exam Request — Required Employee Information

When a company requests a medical examination for an employee, the following information must be provided.

## Exam Type

| Field | Options |
|---|---|
| Exam purpose | Pre-employment / Periodic / Other |
| Job environment | Office / Rig/Field / Workshop/Laboratory |

## Employee Personal Information

| Field | Notes |
|---|---|
| Last Name | In clear capital letters (English) |
| First Name | In clear capital letters (English) |
| Sex | |
| Birth Date | Format: DD/MM/YYYY |
| Nationality | |
| Home Phone | |
| Home Address | |
| ID / Passport Number | |

## Employment Information

| Field | Notes | SLB Label |
|---|---|---|
| Company Name | Name of the employing company | SLB Company |
| Employee ID | Internal company employee number | Employee Company Number (GIN) |
| Job Description | Employee's role or position | Employee Job Description |
| Job Location | Area and country of work | Area and Job Location (Country) |

> **SLB note:** The Employee Company Number is referred to as **GIN** in SLB documentation. Always use GIN when communicating with SLB.

## Consent

The employee must sign a consent statement confirming:

- Medical information will remain strictly confidential.
- Results will be sent to the requesting company's medical department.
- The employee may consult or rectify their medical file by contacting the relevant medical authority.

A **date** and **employee signature** are required.

> **SLB note:** Results are sent to the SLB Medical Department. Employees may rectify their file by writing to Medical Department-SLB Paris (Art. 27-CNIL).
